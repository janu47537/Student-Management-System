# Student Management System (Java)

## Project Description
This is a Java-based Student Management System that allows users to manage student records efficiently.
## Features
 Add Student
 View Students
 Search Student
 Update Student
 Delete Student
 File Handling (Data Persistence)
## Technologies Used
  Java
  OOP Concepts
  File Handling
## How to Run
1. Compile:
   javac Main.java
2. Run:
   java Main
## File Storage
Data is stored in:
students.txt
